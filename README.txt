Инструкция

Файлы в папке:
- index.html — основная страница приложения
- styles.css — стили
- app.js — логика (localStorage, экспорт/импорт)
- manifest.json — веб-манифест (для добавления на главный экран)
- sw.js — простой service worker (для оффлайн)

Как запустить:
1) Самый простой способ: сохранить все файлы в одну папку и открыть index.html в браузере Safari на iPhone (или любом другом браузере).
2) Чтобы установить как приложение: в Safari откройте страницу и выберите "Добавить на экран 'Домой'". Манифест помогает, но для полного PWA лучше разместить на HTTPS-сервере.
3) Чтобы внести изменения в формулы расчёта: нажмите ⚙️ и измените параметры (базовая ставка, множители). Я не знаю точные правила расчетов GSD, поэтому сделал настраиваемую модель:
   - базовая ставка (руб/час)
   - ночная надбавка (множитель для ночных смен)
   - порог сверхурочных (часов в смене)
   - множитель сверхурочных
4) Экспорт/импорт: есть CSV и JSON. JSON удобен для бэкапа/переноса.
